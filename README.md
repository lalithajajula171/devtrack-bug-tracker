# DevTrack - Bug Tracking Management System

## 📌 Project Overview

DevTrack is a web-based Bug Tracking Management System developed using Python Flask and SQLite. This application helps users track, manage, and update software bugs efficiently. Users can register, log in, add bugs, edit bug details, change bug status, search bugs, export reports, and manage their accounts.

## 🚀 Features

- 👤 User Registration
- 🔐 User Login
- 🔑 Forgot Password
- 📊 Dashboard
- ➕ Add Bug
- 📋 View Bugs
- ✏️ Edit Bug
- 🗑️ Delete Bug
- 🔍 Search Bugs
- 🎯 Filter Bugs by Status
- 📄 Export Bugs to CSV
- 🔒 Change Password
- 🚪 Logout
